### retreat

Shows all 10-day Vipassana retreats in your region that haven't open for applications yet.
```
Usage of retreat:
  -all
    	if set, results are unfiltered
  -days string
    	length in days (default "10")
  -from string
    	start date YYYY-MM-DD (default "now")
  -region string
    	region (default "Europe")
  -to string
    	end date YYYY-MM-DD
```
