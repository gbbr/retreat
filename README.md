Shows all 10-day Vipassana retreats in your region that haven't open for applications yet.
```
Usage of retreat:
  -from string
    	from date YYYY-MM-DD (default "now")
  -region string
    	region (default "Europe")
  -student string
    	'old' or 'new' (default "old")
  -to string
    	to date
```
